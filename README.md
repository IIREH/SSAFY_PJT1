  - (아직은 가명입니다.)

- 서비스 설명/ 주요기능

  - 설명 : 문화예술분야 큐레이션 SNS 서비스

  - 주요기능 : 

    -  SNS 기능 (뉴스피드 / 팔로우 / 팔로잉 / 좋아요 / 댓글)

    -  공연정보 제공 기능
