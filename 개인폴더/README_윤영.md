### JSX의 규칙

1. 태그는 꼭 닫혀있어야 한다. 

2. 모든 태그는 2개이상이 된다면 꼭 하나로 감싸져야 한다.

```react
import React from "react";
class App extends React.Component{
  render(){
     return (
      <div>
         <div>
          첫 번째 div입니다.
         </div>
         <div>
          두 번째 div입니다.
        </div>
     </div>
      )       
    }
}

export default App; 
```

- `<div>`가 2개이기 때문에 그 위에 `<div>`로 감싸줘야만 오류가 나지 않는다 .

function Welcome(props) {

}
